A better way to get tasks done

Task Machine is a personal project/task management system. This system will help you do the following to get you projects done:

  • Create detailed tasks
  • Create custom heirarchy of tasks
  • Track completion of the tasks

Customize

Task Machine's text editor will allow you to make your task entries as detailed as you like. You can change font, highlighting and you can even insert images and tables.
